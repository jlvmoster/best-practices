# My Most Frequently Used Bash/Zsh Commands

## ls — List Directory Contents

## echo — Printout Text & Files to Terminal Output

## mkdir — Create a Directory

## grep — Search Strings & File I/O Streams

## cd — Change Directory

## mv - Move or Rename Directory

## rmdir — Remove Directory

## > — Redirect Standard Output (stdout)

## cat — Read, Create, and Concatenate a File

## | — Pipe stdout as Input

## chmod — (Change Mode) Sets File Permissions on Files or Folders

## chown — (Change Owner) Change File Owner and Group
